Tasks selection

According to base exclusion criteria (i.e., the ones that apply to all investigations), excluding the requirement of self-contained tasks, 10 projects remained, from which we selected 1,332 tasks. Finally, from this initial tasks set, we defined two samples: a set of 315 self-contained tasks (our sample) that does not support questions demanding test execution, and another set of 77 tasks (subsample) for dealing with all inquiries, which satisfy the criteria related to tests execution. In sum, we selected 340 unique tasks.

About the subsample, our initial objective was to find ten tasks from ten different projects due to the effort related to environment configuration for test execution. However, three projects do not have ten valid tasks. Thus, we did not assess all tasks per project but just the necessary to reach our goal.
